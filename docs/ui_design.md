# 前端 UI 设计文档（股票基金投资论坛）

## 1. 设计目标

1. 面向“投资讨论”场景，突出信息密度与可读性。
2. 首屏快速找到：市场板块、热榜、关注动态、搜索。
3. 低学习成本，支持新手用户快速发帖与互动。

## 2. 信息架构

### 2.1 导航结构

- 顶部导航：`首页` `市场` `主题` `公司研究` `问答求助` `发布`
- 二级入口：`我的关注` `我的收藏` `个人中心` `消息`

### 2.2 页面结构

1. 首页（Feed + 热榜 + 搜索）
2. 板块页（帖子列表 + 筛选排序）
3. 帖子详情页（正文 + 评论 + 点赞收藏）
4. 发布页（普通帖/长文/实时短帖）
5. 个人中心（资料、偏好、隐私、认证状态）

## 3. 关键页面草图描述

## 3.1 首页

- 左列：板块快捷入口（A股/港股/美股/基金等）
- 中列：帖子流（默认最新，可切换热度）
- 右列：热榜、推荐作者、搜索框

## 3.2 帖子详情

- 顶部：标题、作者、发布时间、所属板块
- 主体：正文、股票代码标签、图片附件
- 互动区：点赞、收藏、评论输入框
- 评论区：一级评论 + 楼中楼回复

## 3.3 发布页

- 字段：板块、标题、内容、股票代码、图片
- 帖子类型：普通 / 长文 / 实时
- 预览区：实时展示发帖效果

## 4. 组件清单

1. `BoardTabs`：板块切换组件
2. `PostCard`：帖子卡片（标题、摘要、互动数）
3. `PostEditor`：发帖编辑器
4. `CommentList`：评论列表
5. `HotRankPanel`：热榜面板
6. `SearchSuggest`：搜索联想下拉

## 5. 交互规则

1. 点赞/收藏为即时反馈，失败时回滚并提示。
2. 发帖成功后跳转至详情页并定位到新帖。
3. 搜索联想在输入 2 个字符后触发。
4. 未登录触发交互动作时，统一弹出登录引导。

## 6. 视觉规范（简版）

- 主色：`#1565C0`（金融蓝）
- 强调色：`#00897B`（增长绿）
- 风险提示色：`#E65100`（警示橙）
- 字体建议：中文 `Noto Sans SC`，英文 `IBM Plex Sans`
- 卡片圆角：8px；阴影轻量，避免干扰阅读

## 7. 响应式方案

1. `>=1200px`：三栏布局（导航/内容/侧边栏）
2. `768px~1199px`：双栏布局（内容 + 可折叠侧栏）
3. `<768px`：单栏，顶部导航改抽屉，发帖按钮固定悬浮

## 8. 与后端接口映射

- 帖子流：`GET /api/v1/content/feed`
- 热榜：`GET /api/v1/content/hot-rank`
- 搜索：`GET /api/v1/content/posts`、`GET /api/v1/content/search/suggest`
- 发帖：`POST /api/v1/content/posts`
- 评论：`POST /api/v1/content/posts/{post_id}/comments`
- 互动：`POST/DELETE /api/v1/content/posts/{post_id}/like|favorite`
