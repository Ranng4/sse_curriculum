# 投资社区系统需求说明

## 1. 用户系统

### 1.1 注册与认证

- 多方式注册：支持手机号、邮箱、第三方账号（微信/微博）注册。
- 分级认证体系：
  - 基础认证：手机/邮箱验证。
  - 实名认证：身份证 + 人脸识别（可选）。
  - 专业认证：上传从业资格、学历证明等（认证后加 V 标识）。
- 投资者适当性评估：专业用户需完成风险评估问卷。

### 1.2 个人资料管理

- 基本信息：昵称、头像、个人简介、投资经验标签。
- 投资偏好：关注领域（A 股 / 港股 / 美股 / 基金等）、风险偏好。
- 成就系统：发帖数、精华帖、影响力值、荣誉勋章。
- 隐私设置：控制信息可见范围。

## 2. 内容系统

### 2.1 板块分类

- 市场讨论区：按市场划分（A 股、港股、美股、期货等）。
- 主题专区：
  - 价值投资专区。
  - 量化投资专区。
  - 基金投资专区。
  - 新股 / 新债讨论。
  - 宏观策略研讨。
- 公司研究专区：按行业、个股进行深度讨论。
- 问答求助区：新手提问、投资解惑。
- 板块管理：所有板块支持动态添加、删除和修改。

### 2.2 发帖与互动

- 多样化发帖：
  - 普通帖子：文字 + 图片 + 基础格式。
  - 长文分析：富文本编辑器，支持图表插入。
  - 投票调研：投资观点调查、市场情绪投票。
  - 实时讨论：盘中简短讨论（类似微博）。
- 内容交互：
  - 点赞 / 收藏 / 转发功能。
  - 多级评论系统（支持楼中楼）。
  - `@` 提及功能。
  - 私信系统（支持文本、图片）。
- 附件支持：上传 PDF、Excel 等分析报告（需审核）。

## 3. 社交与关系系统

### 3.1 关注与粉丝

- 用户互相关注机制。
- 特别关注列表（星标用户）。
- 关注动态推送。

### 3.2 群组功能

- 用户可自建投资主题群组。
- 群组权限管理（公开 / 私密 / 审核加入）。
- 群内专属讨论与资料分享。
