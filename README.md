# 投资社区系统需求说明

# 股票基金投资论坛

## 1 用户系统

### 1.1 注册与认证

- **多方式注册**：支持手机号、邮箱、第三方账号（微信/微博）注册
- **分级认证体系**：
  - 基础认证：手机/邮箱验证
  - 实名认证：身份证 + 人脸识别（可选）
  - 专业认证：上传从业资格、学历证明等（加 V 标识）
- **投资者适当性评估**：专业用户需完成风险评估问卷

### 1.2 个人资料管理

- **基本信息**：昵称、头像、个人简介、投资经验标签
- **投资偏好**：关注领域（A 股 / 港股 / 美股 / 基金等）、风险偏好
- **成就系统**：发帖数、精华帖、影响力值、荣誉勋章
- **隐私设置**：控制信息可见范围

## 2 内容系统

### 2.1 板块分类

- **市场讨论区**：按市场划分（A 股、港股、美股、期货等）
- **主题专区**：
  - 价值投资专区
  - 量化投资专区
  - 基金投资专区
  - 新股 / 新债讨论
  - 宏观策略研讨
- **公司研究专区**：按行业、个股深度讨论
- **问答求助区**：新手提问、投资解惑
- **板块管理**：所有板块可以动态添加、删除和修改

### 2.2 发帖与互动

- **多样化发帖**：
  - 普通帖子：文字 + 图片 + 基础格式
  - 长文分析：富文本编辑器，支持图表插入
  - 投票调研：投资观点调查、市场情绪投票
  - 实时讨论：盘中的简短文字讨论（类似微博）
- **内容交互**：
  - 点赞 / 收藏 / 转发功能
  - 多级评论系统（支持楼中楼）
  - @提及功能
  - 私信系统（支持文本、图片）
- **附件支持**：上传 PDF、Excel 等分析报告（需审核）

## 3 社交与关系系统

### 3.1 关注与粉丝

- 用户互相关注机制
- 特别关注列表（星标用户）
- 关注动态推送

### 3.2 群组功能

- 用户自建投资主题群组
- 群组权限管理（公开 / 私密 / 审核加入）
- 群内专属讨论和资料分享

## 4 信息整合系统

### 4.1 内容聚合

- **精华推荐**：编辑精选 + 算法推荐优质内容
- **热榜排行**：每日 / 每周热门话题、热门股票讨论
- **个性化 feed 流**：基于用户关注和兴趣的内容推送

### 4.2 搜索功能

- **全文搜索**：支持关键词、用户、股票代码搜索
- **高级筛选**：按时间、热度、精华、市场等维度过滤
- **搜索联想**：输入股票代码 / 名称自动补全

## 5 管理运营系统

### 5.1 内容审核

- **自动审核**：敏感词过滤、重复内容检测
- **人工审核**：可疑内容队列、用户举报处理
- **合规检查**：防止荐股、操纵市场等违规内容

### 5.2 用户管理

- 用户行为监控（发帖频率、内容质量）
- 积分与等级系统
- 违规处理流程：警告、禁言、封号

### 5.3 数据分析

- 论坛活跃度统计
- 热门话题分析
- 用户参与度报告
