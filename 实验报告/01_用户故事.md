# 股票基金投资论坛系统用户故事

## 1. 角色定义

| 角色 | 说明 |
| --- | --- |
| 游客 | 未登录用户，可浏览公开内容、查看热榜和搜索公开帖子 |
| 注册用户 | 已完成注册和登录，可发帖、评论、点赞、收藏、关注、私信 |
| 专业用户 | 通过专业认证的用户，可展示认证标识并发布更专业的内容 |
| 管理员 | 负责板块管理、内容审核、用户管理、举报处理和运营统计 |

## 2. 用户系统

| 编号 | 优先级 | 用户故事 |
| --- | --- | --- |

| US-02 | 核心 | 作为一名注册用户，我想使用账号和密码登录系统，以便进入个人账户并使用论坛功能。 |

| US-04 | 核心 | 作为一名注册用户，我想设置投资经验标签和关注市场，以便系统展示更相关的内容。 |
| US-05 | 重要 | 作为一名注册用户，我想填写风险评估问卷，以便平台了解我的风险承受能力。 |
| US-06 | 重要 | 作为一名专业用户，我想提交从业资格或学历证明，以便获得专业认证标识。 |
| US-07 | 扩展 | 作为一名用户，我想设置个人资料可见范围，以便保护隐私信息。 |

## 7. 第一阶段实现范围

本项目已实现并验证的核心范围包括：注册登录、个人资料、适当性评估、板块管理、发帖、评论、点赞、收藏、热榜、搜索联想、关注粉丝、关注流、私信、后台统计、帖子审核和用户管理。短信、第三方 OAuth、人脸识别、附件审核和群组功能作为扩展点保留。
