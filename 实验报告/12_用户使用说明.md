# 用户使用说明

## 1. 产品简介

股票基金投资论坛面向投资交流场景，支持用户注册登录、板块浏览、发帖评论、点赞收藏、关注粉丝、私信沟通、搜索热榜、风险测评和后台管理。

## 2. 用户注册与登录

1. 进入注册页面。
2. 输入手机号或邮箱、密码、昵称和验证码。
3. 注册成功后登录。
4. 登录成功后系统返回 token，前端自动携带 token 调用受保护接口。

## 3. 浏览内容

1. 首页查看推荐内容、最新内容和热门内容。
2. 进入市场讨论区查看 A股、港股、美股、基金等板块。
3. 进入主题专区查看价值投资、量化投资、基金投资等主题。
4. 点击帖子卡片进入帖子详情页。

## 4. 发布和互动

1. 登录后点击发布入口。
2. 选择板块，填写标题、正文、帖子类型和股票代码。
3. 发布成功后进入帖子详情页。
4. 在详情页可发表评论、回复评论、点赞和收藏。
5. 收藏后的帖子可在个人相关页面中查看。

## 5. 搜索与热榜

1. 在搜索框输入关键词、股票名称或股票代码。
2. 系统返回匹配帖子列表。
3. 输入过程中可使用搜索联想。
4. 热榜页面展示互动度较高的帖子。

## 6. 关注与私信

1. 进入其他用户主页。
2. 点击关注后，该用户内容会出现在关注流中。
3. 可查看我的关注和我的粉丝。
4. 点击私信进入会话，发送文本消息。
5. 打开会话后，未读消息会自动标记为已读。

## 7. 个人中心

1. 修改昵称、头像、简介和投资经验标签。
2. 设置关注市场和风险偏好。
3. 完成适当性风险问卷，查看风险等级。
4. 设置资料隐私可见范围。

## 8. 管理员功能

1. 查看论坛统计数据。
2. 管理论坛板块。
3. 查看帖子列表并进行审核、标记或通过。
4. 查看用户列表，对违规用户执行警告、封禁或解封。

## 9. 常见错误

| 错误 | 原因 | 处理方式 |
| --- | --- | --- |
| `401` | 未登录或 token 失效 | 重新登录 |
| `400` | 参数不合法 | 检查必填字段和格式 |
| `404` | 资源不存在 | 刷新列表或检查链接 |
| `409` | 资源冲突 | 避免重复注册或重复操作 |
